# 🚗 Car Rental Booking System

A web-based Car Rental Booking System built using Django. This application allows users to search and book cars, while car dealers can manage their vehicles and track bookings.

---

## 📌 Description

This project provides an easy and efficient way to rent cars online. Users can browse available cars based on location, book them for specific days, and manage their bookings. Car dealers can add cars, manage availability, and monitor earnings.

---

## 🚀 Features

### 👤 Customer

* Register and Login
* Search cars by city
* View available cars
* Book cars
* View past bookings

### 🏢 Car Dealer

* Register and Login
* Add / Edit / Delete cars
* View all cars
* Manage bookings
* Mark orders as complete
* Track earnings

---

## 🛠️ Tech Stack

* Backend: Python (Django)
* Frontend: HTML, CSS, JavaScript
* Database: SQLite
* Authentication: Django User Model

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/your-username/car-rental-system.git
```

2. Go to project folder

```
cd car-rental-system
```

3. Create virtual environment

```
python -m venv venv
```

4. Activate environment

```
venv\Scripts\activate
```

5. Install dependencies

```
pip install -r requirements.txt
```

6. Run migrations

```
python manage.py makemigrations
python manage.py migrate
```

7. Start server

```
python manage.py runserver
```
